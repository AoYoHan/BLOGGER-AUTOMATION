description: Blog Automation Complete Process (자동 프로세스화 가이드)
---
이 업무 흐름은 새롭게 추가된 `run.bat` 파일과 `auto/loop` 명령을 사용하여 블로그 자동화를 "프로세스화"하는 방법을 안내합니다.

### 방법 1: 단계별 수동 (추천)
1. Google Sheets의 **'키워드관리'** 시트에서 키워드와 톤을 입력합니다.
2. `run.bat` 파일을 실행하고 **Option 1 [Generate]**를 선택하여 초안을 생성합니다.
3. 생성 완료 후, Google Sheets의 **'초안검토'** 시트에서 제목, 메타설명, 이미지를 검토합니다.
4. 마음에 들면 **'승인'** 열에 '승인'을 입력합니다.
5. `run.bat`에서 **Option 2 [Publish]**를 선택하여 게시합니다.

### 방법 2: 통합 자동 실행 (속도 향상)
1. 구글 시트에 키워드들을 입력합니다.
2. `run.bat`에서 **Option 4 [Auto]**를 선택합니다.
   * 대기 중인 모든 키워드의 초안을 먼저 생성한 후, 바로 이어서 승인된 모든 초안의 게시 작업을 수행합니다.

### 방법 3: 완전 자동 루프 (손대지 않음)
1. `run.bat`에서 **Option 5 [Loop]**를 선택합니다.
2. 1시간 간격으로 시스템이 알아서 대기 키워드를 처리하고 승인된 글을 게시합니다. (간격 조절은 `main.py` 수정 가능)
3. 이제 당신은 시트에 키워드만 넣어두고, 생각날 때 구글 시트에서 승인 버튼만 누르면 됩니다.

// turbo
### 방법 4: 윈도우 시작 시 자동 실행 설정
1. `Win + R` 키를 누르고 `shell:startup`을 입력하여 시작프로그램 폴더를 엽니다.
2. `run.bat` 파일의 바로가기를 이 폴더에 만듭니다.
3. 이제 PC를 켤 때마다 자동으로 메뉴 창이 나타나 작업 준비가 완료됩니다.
